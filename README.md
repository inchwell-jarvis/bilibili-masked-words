## 设置屏蔽词

1. 打开 B站 PC 端网站。
2. 在右上角，你会看到一个名为 '设置屏蔽词' 的选项。
3. 点击 '设置屏蔽词'，会弹出一个窗口，让你输入要屏蔽的关键词。
4. 输入关键词后，点击 "记录" 或按下回车键，即可保存你的屏蔽词。
5. 插件会在页面变化时自动检查视频内容，以确保不显示包含屏蔽词的视频。
